# Project Overview

## Purpose
This is a comprehensive AI coding workshop template and reference repository for building production-grade AI agents using Pydantic AI. The project demonstrates:

- **PRP Framework**: Product Requirements Prompt methodology for structured AI development
- **Context Engineering**: Four pillars (RAG, Task Management, Memory, Prompt Engineering)
- **Hybrid RAG Agent**: Complete implementation with vector search, hybrid search, and knowledge graph retrieval
- **Development Workflow**: From planning to implementation to validation

## Project Structure
- `PRPs/` - PRP templates, examples, and INITIAL.md for new agent development
- `FullExample/` - Complete RAG agent implementation with ingestion pipeline
- `PromptProgression/` - Evolution from bad prompts to perfect PRPs
- `.claude/` - Slash commands and subagent configurations
- `CLAUDE.md` - Global rules for AI agent development

## Key Workflows
1. **New Agent Development**: INITIAL.md → `/generate-pydantic-ai-prp` → `/execute-pydantic-ai-prp`
2. **Existing Codebase Enhancement**: Codebase investigation → Research → Focused PRPs
3. **Validation**: Multi-tier testing (syntax → types → unit → integration → human review)

## Target Audience
AI engineers, developers building AI agents, and teams adopting structured AI development methodologies.
